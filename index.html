Типичная задача

Добавить на страницу с отчетом возможность отправить csv-версию отчета на почту текущему пользователю

Нужно писать тексты

  1. Надпись на кнопке
  2. Тема и тело письма
  3. Сообщение о том, что письмо отправлено.
  4. Сообщение об ошибке, если не получилось отправить письмо.

Кто будет их писать?

  • Заказчик
  • UX-копирайтер
  • Дизайнер
  • Программист

У программистов есть задачи поважнее

  • Написать оптимальные запросы к БД
  • Разрешить конфликт версий между библиотеками
  • Корректно обрабатывать кодировку файлов
  • Кроссбраузерно сверстать письмо
  • Написать надписи?
